# LC(OS) Manual

## Reading Files
- Use `read /docs/manual.md` to view Markdown with terminal formatting.
- Use `read /docs/manual.txt` for the plain-text variant.

## Batch Scripts
- `run /programs/demo.lcbat` runs the native batch example.
- `run /programs/demo.bat` runs the DOS-style batch example.

## Built-in Commands
- `ls` lists files and directories.
- `read` prints file contents.
- `date` shows the RTC date and time.
